Habilidades

HTML(Linguagem de Marcação de HiperTexto) é uma linguagem de marcação utilizada na construção de páginas na Web.

CSS(Folhas de Estilo em Cascata) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML.

JavaScript (às vezes abreviado para JS) é uma linguagem leve, interpretada e baseada em objetos com funções de primeira classe, mais conhecida como a linguagem de script para páginas web.

O SASS é uma linguagem de extensão do CSS. A sigla significa “Syntactically Awesome Style Sheets” traduzindo ao pé da letra, folhas de estilo com uma sintaxe incrível.
